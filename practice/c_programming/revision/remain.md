# Pending DSA Topics & Questions for Future Practice

Here is a recommended roadmap of topics and questions to practice next for interview and competitive programming preparation.

## 1. Binary Search & Applications
* **Standard Binary Search** - Implement standard $O(\log N)$ search on a sorted array.
* **Lower Bound & Upper Bound** - Find the first and last occurrence of a target element in a sorted array containing duplicates.
* **Search in Rotated Sorted Array** - Find a target element in an array that has been rotated by some pivot.
* **Peak Element** - Find a peak element (an element that is strictly greater than its neighbors) in an unsorted array.

## 2. Linked Lists
* **Singly Linked List Implementation** - Write basic operations for creation, node insertion, node deletion, and traversal.
* **Reverse a Linked List** - Implement reversing a linked list iteratively and recursively.
* **Cycle Detection** - Use Floyd's Cycle-Finding Algorithm (Tortoise and Hare) to detect if a list contains a cycle.
* **Merge Sorted Lists** - Merge two already sorted linked lists into a single sorted list.

## 3. Binary Trees & BST
* **Tree Traversals** - Implement Preorder, Inorder, Postorder traversals (both recursively and iteratively) and Level Order Traversal (BFS).
* **Height & Diameter of a Tree** - Calculate the maximum depth and the longest path between any two nodes in a tree.
* **Binary Search Tree (BST) Operations** - Implement BST search, insertion, and node deletion (handling cases with zero, one, or two children).
* **Lowest Common Ancestor (LCA)** - Find the lowest common ancestor node for two given nodes in a Binary Tree/BST.

## 4. Graphs
* **Graph Representation** - Implement representations using an Adjacency Matrix and Adjacency List.
* **BFS & DFS Traversals** - Write standard Breadth-First Search and Depth-First Search traversals.
* **Cycle Detection** - Detect cycle in directed and undirected graphs.
* **Dijkstra's Algorithm** - Find the shortest path from a single source vertex to all other vertices in a weighted graph.
* **Minimum Spanning Tree (MST)** - Implement Kruskal's or Prim's algorithm.

## 5. Dynamic Programming (DP)
* **DP Introduction** - Implement Fibonacci or climbing stairs using memoization (top-down) and tabulation (bottom-up) to understand subproblems.
* **0/1 Knapsack Problem** - Solve the classic knapsack decision/optimization problem.
* **Longest Common Subsequence (LCS)** - Find the length of the longest subsequence common to two strings.
* **Coin Change Problem** - Find the minimum number of coins needed to make a target change amount.

## 6. Bit Manipulation
* **Bitwise Operations** - Get, set, clear, and toggle specific bits. Check if a number is even/odd or a power of two using bitwise operators.
* **Single Number** - Find the element in an array that appears once while all other elements appear twice (using XOR).
* **Count Set Bits** - Count the number of `1`s in the binary representation of an integer.

## 7. Strings
* **Anagram Check** - Determine if two strings are anagrams of each other (using character counts).
* **Reverse Words** - Reverse the order of words in a given sentence.
* **Pattern Matching** - Implement substring matching using naive search, and study KMP or Rabin-Karp algorithms.

## 8. Backtracking
* **N-Queens Problem** - Place N chess queens on an N×N chessboard so that no two queens threaten each other.
* **Rat in a Maze** - Find all paths from source to destination in a maze grid with blockages.
* **Subsets & Permutations** - Generate all possible subsets (power set) or permutations of a set of numbers.
